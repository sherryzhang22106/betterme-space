# BetterMe Space 更新日志

## v0.2.0 (2026-03-24)

### 新增功能

#### 1. 测评引擎增强
- **数据库驱动评分规则**：评测引擎现在从数据库读取评分规则，无需硬编码
- **动态题目配置**：题目和选项从数据库加载，支持多种题型（单选、量表）
- **结果模板配置化**：每种结果可配置标题、描述、标签

#### 2. 用户记录列表 API
- 新增 `GET /api/records` 接口
- 返回用户所有测评记录
- 支持时间差计算（如"3天前"）

#### 3. 历史记录页面
- 个人中心显示完整测评历史
- 记录卡片展示：测评名称、结果、分数、时间
- 点击可查看详细结果

#### 4. 分享海报功能
- 结果页新增"生成分享海报"按钮
- 使用 html2canvas 生成高清图片
- 支持下载到本地
- 新增"分享到微信"功能

### 数据库更新

运行以下 SQL 更新数据库：

```bash
psql $DATABASE_URL -f lib/assessment-schema.sql
```

新增内容：
- 完整的 MBTI 示例测评（5道题目）
- 4种评测结果类型
- assessment_records 表结构完善

### 技术细节

#### API 变更
- `GET /api/assessments/[id]` - 从数据库读取评测配置
- `POST /api/assessments/[id]/submit` - 数据库评分规则
- `GET /api/records` - 获取用户记录列表
- `GET /api/records/[id]` - 获取记录详情（含完整结果）

#### 前端变更
- ResultPage - 海报生成、微信分享
- UserCenter - 历史记录列表
- QuizPage - Token 认证支持

### 依赖更新
- 新增 `html2canvas: ^1.4.1`
